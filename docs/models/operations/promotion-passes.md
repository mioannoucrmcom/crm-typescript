# PromotionPasses

These attributes are applicable only if the pass plan classification is PROMOTION

## Example Usage

```typescript
import { PromotionPasses } from "crm/models/operations";

let value: PromotionPasses = {

  rewardOfferId: "e2ade504-9b90-92be-0adb-3258475ff695",
  name: "50% Birthday Offer",
  maximumUsage: 1,
};
```
