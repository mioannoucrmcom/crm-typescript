# Recipient

The recipient information.This is the registered/unregistered contact who will be redeeming the pass. 

## Example Usage

```typescript
import { Recipient } from "crm/models/operations";

let value: Recipient = {
  name: "John Smith",
  mediumType: "EMAIL",
  mediumValue: "j.smith@crm.com",
  countryCode: "GNB",
  language: "EN",
  registeredContact: {
    contactId: "acaae121-b4af-4663-8646-47bb541c971d",
    walletId: "4613eb36-e5e8-46e8-9517-b1a31e86c89c",
  },
};
```
