# DeliveryMethods

Defines how delivery order was/will be provided - comma separated e.g. DIRECT_DELIVERY, COURIER_SERVICE

## Example Usage

```typescript
import { DeliveryMethods } from "crm/models/operations";

let value: DeliveryMethods = "DIRECT_DELIVERY";
```

## Values

```typescript
"DIRECT_DELIVERY" | "COURIER_SERVICE" | "PICKUP_POINT"
```
