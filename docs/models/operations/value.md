# Value

The pass plan value information, applicable only for pass plans with classification of GIFT or TOP_UP

## Example Usage

```typescript
import { Value } from "crmcom/models/operations";

let value: Value = {
  type: "FIXED",
  fixedAmount: 100,
  minimum: 10,
  maximum: 50,
  currencyCode: "EUR",
};
```
