# FulfilmentFlows

Settings that determine the order’s behaviour and how it will be fulfilled by the business and/or the merchants/service provides in the Business Network

## Example Usage

```typescript
import { FulfilmentFlows } from "crm/models/operations";

let value: FulfilmentFlows = {
  fulfilmentFlow: "SINGLE_MERCHANT",
  allowDeliverySelection: true,
};
```
