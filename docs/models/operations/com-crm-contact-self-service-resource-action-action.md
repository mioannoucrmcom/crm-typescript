# ComCrmContactSelfServiceResourceActionAction

The action to be performed on the contact relatioship status
 * `ACCEPT` - Accept the invitation to be part of a community
 * `REJECT` - Accept the invitation to be part of a community


## Example Usage

```typescript
import { ComCrmContactSelfServiceResourceActionAction } from "crm/models/operations";

let value: ComCrmContactSelfServiceResourceActionAction = "ACCEPT";
```
