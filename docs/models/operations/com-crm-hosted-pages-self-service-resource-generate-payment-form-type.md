# ComCrmHostedPagesSelfServiceResourceGeneratePaymentFormType

The type will be utilised by CRM.COM for back-end processes. Payment forms can be used to either make a Payment or perform a Top-up. Use the “type” to differentiate the purpose of loading the payment form.

```typescript
import { ComCrmHostedPagesSelfServiceResourceGeneratePaymentFormType } from "crmcom/models/operations";

let value: ComCrmHostedPagesSelfServiceResourceGeneratePaymentFormType =
  "PAYMENT";
```
